# Shadow Client Clean

Clean rebuild for a lightweight Fabric client mod targeting Minecraft 1.21.11.

## Current features

- Right Shift opens Shadow Client module menu
- Minimal gray aesthetic
- Toggle cards for FPS, Ping, CPS, Keystrokes, Reach Display, TierTagger, Menu Blur, Motion Blur, Armor HUD, Potion HUD
- No cheat modules and no reach modification
- GitHub Actions workflow included to build the jar online

## Build on GitHub

1. Upload all files in this folder to your GitHub repository.
2. Open the Actions tab.
3. Run `Build Shadow Client Clean`.
4. Download the `shadow-client-clean-jar` artifact.
5. Put the jar into `D:\shadow\mods`.
6. Set Minecraft Launcher Fabric 1.21.11 game directory to `D:\shadow`.
